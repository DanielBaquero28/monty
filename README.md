## Setup
####Monty interpreter
This command will be used to compile the monty interpreter:
``gcc -Wall -Werror -Wextra -pedantic *.c -o monty``
Please use gcc-4.8

####Brainfuck programs
To install the brainfuck interpreter:
``sudo apt_get install bf``
